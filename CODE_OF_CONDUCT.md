# Code of Conduct

This project adopts the **[Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**
(v2.1) as its code of conduct.

In short: be respectful, assume good intent, welcome newcomers, and keep discussion focused on the
work. Harassment of any kind is not tolerated.

## Reporting

Report unacceptable behavior privately to the maintainer at **hingneshubham@gmail.com**. Reports are
handled confidentially. For security issues, follow [SECURITY.md](SECURITY.md) instead.
